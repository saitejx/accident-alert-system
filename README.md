# Accident Detection and Alert System
A web application that detects accidents and sends alerts to parents, hospitals, and police.

## Features
- Detects accident intensity
- Sends SMS alerts
- Sends email notifications
- Alerts hospitals and police based on severity

## Installation
1. Clone the repository
2. Install dependencies
3. Run the application

 ## Technologies
- Python
- Flask
- HTML/CSS
- JavaScript
- Twilio API
