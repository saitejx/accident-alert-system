# Accident Detection & Alert System 🚨

A web-based accident detection and emergency alert system.

## Features
- Live location detection
- Interactive map with Leaflet
- Accident severity selection
- Emergency email alerts
- Automated phone call alert using Twilio
- 10-second confirmation timer

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- Twilio API
- Leaflet Maps

## How to Run

1. Clone the repo

git clone https://github.com/saitejx/accident-alert-system.git

2. Install dependencies

npm install

3. Create `.env`

TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE=

4. Run server

node server.js

5. Open web app

http://127.0.0.1:5500
